## How to Run

### Java

To compile and run a Java file:

```sh
javac FileName.java
java MainClassName
```

### Python

To run a Python script:

```sh
python script_name.py
```

### C++

To compile and run a C++ file:

```sh
g++ file_name.cpp -o output_name
./output_name
```

### SQL

To execute a SQL script, you can use a command-line client for your database system. Here is an example using `mysql`:

```sh
mysql -u your_username -p your_database_name < script_name.sql
```
